第一章 解码科学上网：不止于翻墙

科学上网（又称"翻墙"）绝非简单的技术规避，而是一套完整的网络自由解决方案。其核心在于通过加密隧道技术，将用户的网络请求"伪装"成可被审查系统接受的数据包。这种技术演化经历了三个阶段：

  1. 原始代理时代（2000-2010）：简单的HTTP代理服务器，如同给网络流量戴上面具
  2. 加密协议革命（2010-2018）：OpenVPN、L2TP等协议的出现，让数据穿上防弹衣
  3. 混淆技术时代（2018至今）：Shadowsocks、V2Ray等工具能将加密流量伪装成正常HTTPS流量

值得注意的是，2022年剑桥大学的研究显示，科学上网用户中68%是为获取学术资源，而非单纯访问社交平台。这颠覆了公众对该技术"主要用于娱乐"的刻板印象。

FAQ

如何在 V2Ray 中自定义路由规则？
在 V2Ray 中可以通过修改 config.json 文件中的 routing 部分来自定义路由规则。用户可以根据 IP 段、域名或地理位置来决定流量的去向，从而实现自动分流或访问加速的效果。
PassWall2 在路由器上使用 TUN 模式会有什么影响？
启用 TUN 模式会拦截并转发路由器上全部或选定流量，这会增加路由器 CPU 与内存负担，可能影响路由器其他服务表现。建议在性能足够的设备上启用，或仅对特定设备/端口进行分流。
Nekoray 节点订阅如何实现自动更新？
用户可添加订阅 URL，Nekoray 会定期自动下载并更新节点列表。保持节点最新可减少手动维护，提高节点可用性和网络连接稳定性，确保科学上网体验顺畅。
Matsuri 节点延迟排序功能如何使用？
Matsuri 会定期测速节点延迟，并结合策略组自动选择延迟最低节点。用户无需手动切换即可保持高速访问，优化网页、视频和游戏体验，提高网络稳定性。
Nekobox 是否支持多客户端同时管理节点？
支持。Nekobox 可以独立管理多个客户端，如 Shadowsocks 和 Clash 节点，互不干扰。结合策略组，可实现多客户端流量分流和网络优化，提高便利性和效率。
V2Ray 的流量分流如何与防火墙配合使用？
通过 V2Ray 的路由规则，可将指定流量通过代理出口，而防火墙限制其他流量直连。结合防火墙规则可以实现对内部网络访问和外部代理流量的精准控制。
Surfboard 与 Shadowrocket 有什么区别？
两者都是 iOS 上的代理工具。Shadowrocket 功能更全面、更新更频繁，而 Surfboard 以轻量和快速见长。若只需简单分流与节点切换，Surfboard 足以满足日常使用。
SSR 的 UDP 转发功能需要注意什么？
SSR 的 UDP 转发需要服务端和客户端同时开启，同时确保服务器防火墙允许 UDP 通信。UDP 流量不可靠，可能在高丢包网络中导致连接中断，需要测试稳定性。
PassWall2 能运行在哪些设备上？
PassWall2 是基于 OpenWRT 的插件，只要路由器支持该固件，就能安装运行。常见型号如小米路由、斐讯、软路由都可以使用，通过 Web 界面操作，适合家庭网络使用场景。
Nekoray 节点订阅自动更新如何实现？
用户添加订阅 URL 后，Nekoray 会定期下载最新节点列表并替换旧节点。保持节点列表最新可减少手动维护，保证节点可用性和网络连接稳定性，提升使用体验。
